El problema

En la actualidad existen una masiva cantidad de personas que se alejan de sus ciudades o lugar natal ya sea por cuestiones de trabajo, viajes, independizarse o uno de los más comunes últimamente estudios, enfrentándose así a tener que buscar un nuevo alojamiento.

El proyecto

Teniendo en cuenta lo mencionado anteriormente surge la pregunta ¿Cómo se puede mejorar la búsqueda de vivienda? Este proyecto pretende abordar esta problemática y poder desarrollar un sistema en el que se pueda realizar una búsqueda más ágil y sencilla de posibles viviendas.

TECNOLOGÍAS Y CONOCIMIENTOS EMPLEADOS

HTML

HTML, acrónimo en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado utilizado en la creación de páginas web, es por eso que decidimos emplearlo para elaborar nuestro proyyecto.

CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (en-US) (incluyendo varios lenguajes basados en XML como SVG, MathML o XHTML).

BOOTSTRAP

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web. Esta fue de utilidad a la hora de guiarnos y mostrarnos todas las forams posibles de estructurar y configurar nuestra pagina.

Microsoft Azure

Microsoft Azure es una plataforma de computación en la nube creado por Microsoft para construir, probar, desplegar y administrar aplicaciones y servicios mediante el uso de sus centros de datos.
